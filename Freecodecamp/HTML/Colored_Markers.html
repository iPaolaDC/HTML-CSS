CSS Color Markers


RGB Colors

Background-color:rgb(red, green, blue);


RGBA Colors

Background-color:rgb(red, green, blue, Alpha);


Secundary Colors

Background-color:rgb(255, 255, 0); → Amarillo

Background-color:rgb(0, 255, 255); → Cian

Background-color:rgb(255, 0, 255); → Magenta


Tertiary Colors

Background-color:rgb(255, 127, 0); → Naranja

Background-color:rgb(0, 255, 127); → Verde Primavera

Background-color:rgb(127, 0, 255); → Violeta

Background-color:rgb(127, 255, 0); → Verde Chartreuse

Background-color:rgb(0, 127, 255); → Azure

Background-color:rgb(255, 0, 127); → Rosa


Colores con valor Hexadecimal

Background-color:#00FF00 → Hexadecimal

Modelo HSL, siglas de hue, saturation y lightness (matiz, saturación y luminosidad)

Background-color:hsl (240 100% 50%) → Hexadecimal

La función linear-gradient

Background-color:linear-gradient(gradientDirection, color1, color2, ...);


Sombras

box-shadow:offsetX offsetY blurRadius color;
